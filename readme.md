# LP Games

A small collection of one-more-go browser games about public money, power, and where trust leaks out. No install, no sign-up — each game is a single self-contained `index.html`.

**▶ Play: https://rcko.github.io/LPGames/**

## Games

### Follow the Money
- **[Audit the Ledger](https://rcko.github.io/LPGames/audit-the-ledger/)** — Scan each page of the ledger and flag the skims before the audit window closes.
- **[Tug of the Dollar](https://rcko.github.io/LPGames/tug-of-the-dollar/)** — Rapid-tap to haul a taxpayer dollar toward public infrastructure and throw off the skimming grabs.

### Revolving Door
- **[Musical Chairs](https://rcko.github.io/LPGames/musical-chairs/)** — Stop the wheel on the Cooling-Off seat, not another revolving-door job.
- **[Sort the Suits](https://rcko.github.io/LPGames/sort-the-suits/)** — File each figure through the right door before they slip out to the street.

### Catch and Release
- **[Beat Patrol](https://rcko.github.io/LPGames/BeatPatrol/)** — Walk the beat, intercept the offenders, watch them come back to the same shop.
- **[The Docket](https://rcko.github.io/LPGames/TheDocket/)** — Work a day's list. Read the file, make the call: Remand or Bail.
- **[Whack-a-Crook](https://rcko.github.io/LPGames/WhackACrook/)** — Tap the figures before they duck away. Catch one, and the case is back up somewhere else.

## Hosting

Served as static files via GitHub Pages from the `main` branch. The root [`index.html`](index.html) is the landing page; each game lives in its own folder.
